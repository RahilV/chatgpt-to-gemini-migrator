Quick start

Get up and running in 4 simple steps

1

Export from ChatGPT

Go to ChatGPT ‚Üí Settings ‚Üí Data controls ‚Üí Export data. Download the ZIP file you receive via email.

2

Place the file

Copy conversations.json to the input folder.

3

Run the migration

python migrate_to_gemini.py

This converts your conversations to multiple formats.

4

View your conversations

python view_conversations.py

Opens a beautiful viewer at http://localhost:8000
